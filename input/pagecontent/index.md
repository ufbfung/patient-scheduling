### Introduction

This FHIR IG was created as an resource for the [Patient Scheduling track](https://confluence.hl7.org/spaces/FHIR/pages/403865700/2026+-+01+Patient+Scheduling) in preparation for the [2026 January HL7 FHIR Connectahon 41](https://hl7.my.site.com/LightningMemberPortal/s/lt-event?id=a1YVM00000O7lfV2AR). 

### Scope and purpose
This FHIR IG is intended to be aspirational and define use cases that aim to streamline the entire end-to-end process for patient scheduling. 

| Use case | What it enables |
|--|--|
| [Provider discovery](use-cases-provider-discovery.html) | Find providers based on specialty, location, language, availability, and other constraints |
| [Calendar sync](use-cases-calendar-sync.html) | Discover and synchronize available appointment slots |
| [Task assignment](use-cases-task-assignment.html) | Assign scheduling tasks to caregivers or care team members |
| [Transportation](use-cases-transportation.html) | Coordinate transportation to and from appointments |
